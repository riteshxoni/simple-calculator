# Simple Calculator

This is a basic calculator program implemented in Java.

## Features

- Addition
- Subtraction
- Multiplication
- Division

## Requirements

- Java Development Kit (JDK) installed.

## How to Run

1. Compile the Java file:
   ```sh
   javac Calculator.java
